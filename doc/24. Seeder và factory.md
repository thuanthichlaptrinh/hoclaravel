# Seeder và Factories

## Factory

-   Factory dùng để định nghĩa cách sinh dữ liệu giả cho một Model
-   Factory KHÔNG tự ghi dữ liệu vào DB, nó chỉ mô tả cấu trúc dữ liệu.
-   Laravel dùng thư viện Faker để random dữ liệu
-   Tạo factory:

```bash
php artisan make:factory CommentFactory --model=Comment
```

## Seeder

-   Seeder dùng để ghi dữ liệu vào database

-   Lệnh tạo seeder:

```bash
php artisan make:seeder ten_seeder
```

-   Lệnh chạy seeder:

```bash
php artisan db:seed

php artisan db:seed --class=ten_file
```
