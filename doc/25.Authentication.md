# Authentication - Xác thực đăng nhập

1. Laravel Breeze -> đơn giản nhất và nhanh
2. Laravel Jetstream -> đầy đủ hơn, 2FA, profile
3. Laravel UI -> cho các dự án cũ, hỗ trợ Balde, React...

**Laravel Breeze:**

-   **Lệnh cài đặt**

    ```bash
    composer require laravel/breeze --dev
    php artisan breeze:install
    ```

-   **Thư mục**:

    -   routes/auth.php
    -   controllers/Auth
    -   recources/views/auth

-   **Thành phần chính của Authentication**

    -   **_Guard_**: định nghia cách laravel xác thực người dùng (phương thức xác thực người dùng được lưu trong config)
        ```php
        'guards' => [
            'web' => [
                'driver' => 'session', // cách xác thực là dùng driver Auth::guard('web')->user()
                'provider' => 'users', // giúp laravel biết được lấy user từ bảng nào, model nào
            ],
        ],
        ```
    -   **_Provider_**: xác định nguồn dữ liệu người dung (giúp laravel biết được là lấy user từ bảng nào và từ model nào)

        ```php
        'providers' => [
            'users' => [
                'driver' => 'eloquent',
                'model' => env('AUTH_MODEL', App\Models\User::class),
            ],
        ]
        ```

    -   **_User Prodiver Model_**:
    -   **_Middleware_**:
        -   auth (chặn truy cập nếu chưa đăng nhập)
        -   guest (ngăn không cho truy cập nếu đã đăng nhập)
    -   **_Session & cookie_**: login thành công -> user_id được lưu vào session -> session_id lưu vào trong cookie của trình duyệt -> gửi request -> đọc session trong cookie -> xác định user
