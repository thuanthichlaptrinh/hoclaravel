# Route trong Laravel

**Route (tuyến đường)** trong Laravel là cách bạn định nghĩa **URL nào sẽ chạy chức năng nào**.

-   **Ví dụ**: Khi người dùng truy cập một địa chỉ (ví dụ: `/tin-tuc`), Laravel sẽ so khớp nó với các route bạn đã định nghĩa, rồi chạy đoạn code tương ứng (controller, closure, view...).

## Các phương thức Route phổ biến

| Câu lệnh                             | Ý nghĩa                                                               |
| ------------------------------------ | --------------------------------------------------------------------- |
| `Route::get()`                       | Truy cập bằng trình duyệt (phương thức GET - mặc định)                |
| `Route::post()`                      | Gửi dữ liệu từ form hoặc AJAX (phương thức POST)                      |
| `Route::put()`                       | Cập nhật dữ liệu (phương thức PUT)                                    |
| `Route::patch()`                     | Cập nhật một phần dữ liệu (phương thức PATCH)                         |
| `Route::delete()`                    | Xoá dữ liệu (phương thức DELETE)                                      |
| `Route::match(['get', 'post'], ...)` | Chấp nhận nhiều phương thức HTTP được chỉ định                        |
| `Route::any(...)`                    | Chấp nhận tất cả các phương thức HTTP (ít dùng, cẩn thận khi sử dụng) |

**Ví dụ khai báo route cơ bản:**

```php
Route::get('/tin-tuc', function () {
    return view('news.index');
});

Route::post('/tin-tuc/store', [NewsController::class, 'store']);

Route::put('/tin-tuc/{id}', [NewsController::class, 'update']);

Route::delete('/tin-tuc/{id}', [NewsController::class, 'destroy']);
```

**Truyền tham số vào route**

```php
Route::get('/user/{id}', function($id) {
    return 'Người dùng có id là: ' . $id;
});

Route::get('/user/{id}/{type}', function($id, $type) {
    return 'Người dùng có id là: ' . $id . '</br>' . 'Type: ' . $type;
})
```

**Route gọi đến Controller**

```php
Route::get('/home', [HomeController::class, 'index']);
```

**Route nhóm và tiền tố**

```php
Route::controller(HomeController::class)->group(function() {
    Route::get('/home', 'index');
    Route::get('/home2', 'index2');
    Route::get('/home3', 'index3');
});

Route::prefix('admin')->name('admin.')->group(function() {
    Route::get('/user/{id}', function($id) {
        return 'Route User - Admin: ' . $id;
    });

    Route::get('/dashboard', function() {
        return 'Route Route dashboard - Admin';
    });
});
```
