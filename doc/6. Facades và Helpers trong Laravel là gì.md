# Facades và Helpers trong Laravel

## 1. Facade là gì? (Hiểu đơn giản)

Facade giống như **cửa sổ truy cập nhanh** đến các chức năng lớn bên trong Laravel (gọi là service), mà không cần viết code dài dòng.

| Facade    | Chức năng              |
| --------- | ---------------------- |
| `DB`      | Truy vấn cơ sở dữ liệu |
| `Auth`    | Xác thực người dùng    |
| `Log`     | Ghi log                |
| `Cache`   | Quản lý cache          |
| `Session` | Quản lý session        |
| `Mail`    | Gửi email              |

**Ví dụ sử dụng:**

```php
DB::table('users')->get();
Auth::check();
Log::info('Có người đăng nhập');
Cache::put('key', 'value', 60);
```

## 2. Helper là gì? (Hiểu đơn giản)

Helper là những hàm PHP viết sẵn giúp công việc trong Laravel trở nên siêu nhanh và gọn gàng. Bạn có thể gọi chúng ở bất kỳ đâu mà không cần dùng facade hay namespace dài.

route('home'); // Lấy URL theo tên route đã định nghĩa
url('/admin'); // Trả về URL đầy đủ (ví dụ: https://example.com/admin)
view('welcome'); // Render và trả về view welcome.blade.php
config('app.name'); // Truy xuất giá trị config (ở đây là tên ứng dụng)
app()->make('service'); // Lấy instance của một service từ container
