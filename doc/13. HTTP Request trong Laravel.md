# HTTP Request trong Laravel

## HTTP Request

Khi người dùng truy cập website hoặc gửi dữ liệu qua form, trình duyệt sẽ gửi **HTTP Request** đến server.  
Laravel xử lý request này thông qua class `Illuminate\Http\Request`.

### Các chức năng chính của Request

-   Lấy input từ form (POST)
-   Lấy query từ URL (GET)
-   Kiểm tra method (GET/POST/PUT/DELETE)
-   Xử lý file upload
-   Validate dữ liệu đầu vào

### Bảng các phương thức thường dùng

| Mục tiêu            | Cách dùng                       |
| ------------------- | ------------------------------- |
| Lấy form input      | `$request->input('ten_truong')` |
| Lấy query URL       | `$request->query('tu_khoa')`    |
| Lấy toàn bộ dữ liệu | `$request->all()`               |
| Kiểm tra tồn tại    | `$request->has('ten_truong')`   |
| Kiểm tra method     | `$request->method()`            |
| Upload file         | `$request->file('tenfile')`     |
| Validate input      | `$request->validate([...])`     |
