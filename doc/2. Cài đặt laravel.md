# Cài đặt Laravel

-   Yêu cầu hệ thống để chạy Laravel:

    -   Hệ điều hành: Linux (Ubuntu, CentOS...), macOS hoặc Windows đều được
    -   Máy chủ web: Apache, Nginx hoặc Laravel Sail (Docker-based dev environment)
    -   Phần mềm: PHP, Composer, MySQL, Node.js & NPM
    -   Laravel yêu cầu các extensions sau được bật trong PHP: CMath, Ctype, Fileinfo, JSON, Mbstring, OpenSSL, PDO, Tokenizer, XML (php -m)

-   Cách cài đặt Laravel Project:
    -   $HOME/.composer/vendor/bin
        echo 'export PATH="$PATH:$HOME/.composer/vendor/bin" >> ~/.bashrc
    -   Cách 1: Cài Laravel bằng Composer (cách truyền thống)
        composer create-project laravel/laravel ten-du-an
    -   Cách 2: Cài Laravel bằng Laravel Installer (nhanh hơn)
        -   Bước 1: Cài Laravel Installer
            composer global require laravel/installer
        -   Bước 2: Tạo dự án Laravel laravel new ten-du-an
        -   Lệnh chạy:
            php artisan serve
