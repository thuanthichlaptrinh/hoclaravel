# HTTP Response trong Laravel

## HTTP Response là gì?

-   Khi người dùng gửi request đến Laravel, ứng dụng sẽ xử lý và trả về phản hồi (HTTP Response)

-   Phản hồi có thể là:

    -   Một chuỗi HTML/text
    -   Một view
    -   Một đối tượng JSON
    -   Một file tải về
    -   Một chuyển hướng (redirect)

-   Laravel sử dụng class: IIluminate\Http\Response

## Helper: Response() là gì?

-   reponse() là một helper function trong laravel, bạn tạo đối tượng HTTP Response một cách linh hoạt hơn thay vì chỉ return chuỗi hoặc view.
