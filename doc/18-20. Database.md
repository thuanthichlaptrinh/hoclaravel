# Database trong laravel

-   SQL thuần: thủ công
-   Query Builder: thủ công, gọn hơn, an toàn hơn SQL thuần
-   ORM: hướng đối tượng, có nhiều tính năng hơn và nhanh hơn
    ORM: Eloquent (Object Relational Mapping) tích hợp sẵn trong Laravel.
    -   Object: Là các class (Model) trong PHP mà bạn code.
    -   Relational: Là các bảng dữ liệu (table) trong cơ sở dữ liệu quan hệ (MySQL, PostgreSQL...).
    -   Mapping: Là quá trình ánh xạ dữ liệu giữa Object → Table.
    -   Mỗi Model đại diện cho một bảng trong DB.
    -   Moi instance (object) của Model là một hàng (record) trong bảng đó.
    -   Eloquent cho phép bạn thao tác với bảng dữ liệu bằng cú pháp hướng đối tượng thay vì viết SQL trực tiếp.
