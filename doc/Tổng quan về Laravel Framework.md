# LARAVEL FRAMEWORK

1. Laravel là gì?
   Laravel là một framework PHP mã nguồn mở, được thiết kế để giúp lập trình viên xây dựng các web application hiện đại, nhanh chóng và hiệu quả.

    - Ra mắt bởi Taylor Otwell năm 2011.
    - Dựa trên mô hình MVC (Model - View Controller).
    - Hướng đến clean code, dễ bảo trì, và phát triển nhanh chóng.

    Laravel không chỉ là framework PHP – mà là hệ sinh thái hoàn chỉnh với:

    - Routing, Template Engine (Blade), ORM (Eloquent), CLI Tool (Artisan)...
    - Hàng trăm package mở rộng mạnh mẽ.

2. Tại sao nên chọn Laravel?

-   Cú pháp rõ ràng, dễ học
    -   Laravel có cú pháp gần gũi, dễ đọc, giúp lập trình viên mới tiếp cận dễ dàng.
    -   Hạn chế code phức tạp nhờ sử dụng các cú pháp gọn gàng và cú pháp ngữ nghĩa.
-   Hỗ trợ mô hình MVC chuẩn
    -   Laravel áp dụng chuẩn MVC (Model View Controller).
    -   Giúp tách biệt giao diện, logic xử lý và dữ liệu - dễ bảo trì, mở rộng về sau.
-   Tích hợp sẵn nhiều công cụ mạnh mẽ
    -   Artisan CLI: Tạo nhanh controller, model, migration...
    -   Blade: Template engine mạnh mẽ, dễ sử dụng.
    -   Eloquent ORM: Quản lý cơ sở dữ liệu như làm việc với đối tượng.
    -   Validation, Middleware, Auth, Mail, Queue tích hợp sẵn.
-   Khả năng mở rộng và scale cao
    -   Laravel có thể chạy từ dự án nhỏ đến hàng triệu người dùng.
    -   Hỗ trợ cache (Redis), queue, broadcasting, Horizon, Octane...
    -   Dễ dàng tích hợp với Docker, Laravel Forge, Vapor để deploy và scale.
-   Bảo mật tốt
    -   Tích hợp sẵn các biện pháp chóng
        -   CSRF (Cross-site request forgery)
        -   XSS (Cross-site scripting)
        -   SQL Injection
    -   Cơ chế Hash password, Auth guard, Policy giúp bảo vệ hệ thống.

3. Ứng dụng

-   Laravel phù hợp với các dự án sau:
    -   Hệ thống quản lý doanh nghiệp: (CRM, ERP, HRM, ...)
    -   Cổng thanh toán, hệ thống thương mại điện tử.
    -   Hệ thống API Backend cho mobile app.
    -   Nền tảng học giáo dục trực tuyến (LMS).
    -   Website tin tức, blog, mạng xã hội nhỏ.
