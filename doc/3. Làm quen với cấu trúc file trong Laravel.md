# Làm quen với cấu trúc file trong Laravel

-   app:
    -   Model:
    -   Controller
    -   Providers: nạp cấu hình
    -   Middleware
-   bootstrap: khởi động ứng dụng
-   config: cấu hình hệ thống
-   database:
    -   factories: tạo dữ liệu giả để test dự án
    -   migrations: thao tác với database
    -   seeders: thêm dữ liệu mẫu
-   public: webroot - file truy cập từ trình duyệt
-   resources: giao diện + assets
-   routes: định tuyến url cho ứng dụng
    -   web.php: route cho website
    -   console.php: lệnh artisan tùy chỉnh
    -   api.php: định tuyến route API
-   storage:
-   vendor: chứa các thư viện bên ngoài mà chúng ta cài đặt
-   file .env: cấu hình môi trường
-   artisan: command-line tool laravel
-   package.json: quản lý các gói js (víte, mix, vue, ...)
-   composer.json: file cấu hình thư viện cài đặt
-   composer.lock: ghi lại các phiên bản cụ thể mà chúng ta cài đặt
