# Vòng đời Request Lifecycle trong Laravel 12.md

Trình duyệt -> gửi request
-> public/index.php: load file autoload + gọi file bootstrap/app và giao cho nó xử lý request
-> bootstrap/app.php:
-> routes/web.php hoặc routes/api.php
-> middleware: bước trung gian xử lý kiểm tra request
-> controller: nhận request và xử lý
-> models: lấy ra dữ liệu
-> response cho trình duyệt: view(), json, ...
